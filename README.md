# Analytics with Pokémon - Kaggle Dataset
This used a practice Kaggle dataset aimed at learning some of the visualisations presented using the Pokémon dataset by [Alberto Barradas](https://www.kaggle.com/abcsds/pokemon). Please visit http://rpubs.com/the_roth/PokemonAnalytics for the analysis.

Some of the R code is shown in the above file. If you'd like to see the whole raw markdown and R code, click on the Type1Plots.Rmd file and select "Raw".
